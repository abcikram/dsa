| JavaScript Map  | C++ unordered_map                         |
| --------------- | ----------------------------------------- |
| `map.set(k,v)`  | `mp[k] = v`                               |
| `map.has(k)`    | `mp.find(k) != mp.end()`                  |
| `map.get(k)`    | `mp[k]` *(only after checking existence)* |
| `map.delete(k)` | `mp.erase(k)`                             |
| `map.size`      | `mp.size()`                               |
